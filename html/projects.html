Snakers

Snakers es una aplicación web full-stack. Ha sido desarrollada utilizando la plataforma Eclipse para toda la programación y el servidor Tomcat para el despliegue. Para el almacenamiento de datos, se creó una base de datos en MariaDB. Se han utilizado lenguajes como HTML5, CSS, JAVASCRIPT, J2EE, SQL y PHP.

La aplicación se desarrolló como proyecto para la asignatura sobre Programación Web del grado de Ingeniería Multimedia, con la que recibí la Matrícula de Honor.
Se han creado desde cero tanto la web como su identidad visual.

Enlace al proyecto: https://github.com/aljanue/Snakers
