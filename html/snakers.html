Con un naming sencillo a la vez que impactante que surge a través de un juego de palabras entre snake🐍 y sneakers👟 y un logo acorde al nombre, llega Snakers un e-commerce full-stack💻 que se desarrolló como proyecto para la asignatura Programación Hipermedia, con el que conseguí la Matrícula de Honor ⭐

SNAKERS🐍 presenta un diseño cuidado, con colores neutros ⚪⚫ y una interfaz elegante, donde comprar es más fácil que nunca gracias a los pop-ups automáticos de confirmación. Simplemente inicia sesión, añade el producto al carrito🛒 y finaliza la compra.
Antes de finalizar tu compra, el servidor analizará si todavía quedan unidades de tu producto para verificar que se puede realizar la venta ✅

En cuanto a las tecnologías, destacar que se desarrollo utilizando el entorno de desarrolló Eclipse🌓 y Tomcat como servidor local. Para el FRONT-END se utilizó Bootstrap y lenguajes de programación como HTML, CSS y JavaScript.
Para el BACK-END, se utilizó una Base de datos SQL en MariaDB para las consultas🔎 y Java Servlet, uniendo el FRONT y el BACK mediante JavaServer Pages (jsp) 📄

También se desarrolló mediante PHP la página de Administración. Con verificación de usuario con derechos de admin obligatorios y todas las funcionalidades para la gestión de productos, pedidos, usuarios y mensajes de contacto necesarias para poder manipular la base de datos de forma externa, para usuarios sin conocimiento en la materia, pudiendo eliminar o agregar productos, desactivar usuarios, entregar o cancelar pedidos, etc.
